Audio

    Common attributes to control playback of audio

  • autoplay: Boolean; use sparingly
  • controls: provides playback controls to user
  • loop: Boolean; will play audio over and over
  • muted: Boolean; can use this to mute audio on page load
  • src: link to the audio source file
  • volume: Range of 0.0 (silent) to 1.0 (loudest)

Video

    Common attributes to control playback of video.

  • autoplay: Boolean; use sparingly
  • controls: provides playback controls to user
  • height: MUST be an absolute value; no percentages!
  • muted: Boolean; can use this to mute audio on page load
  • poster: use an alternate image in place of the first frame before playback has been initialized
  • src: link to the video source file
  • volume: Range of 0.0 (silent) to 1.0 (loudest)
  • width: May be an absolute value or percentages
